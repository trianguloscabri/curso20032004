Problema 115.- Ángulos

28. Calcular los ángulos de un triángulo en los casos siguientes:

1º.- < B = 2  (< A)   y  < C = 3 (< A).                            

2º.- < A = 3 (< B)    y  < C= 45º

3º.- < A + < B = 90º  y < C - < A = 30º 

4º.- < A - < B = 45º  y  < A - < C = 30º

Bruño (1958). Geometría. Curso Superior. Solucionario.  Editorial Bruño. Madrid.

Solución de la alumna Maite Peña Alcaraz del Colegio Portaceli de Sevilla:

 

Utilizando que los ángulos de un triángulo suman 180º; entonces,

1.- A+B+C=1A+2A+3A=6A=180º; luego A=30º, B=60º, C=90º

2.- A+B+C=3B+B+45=180; 4B=135, B=33,75º,  A=101,25º

3.- A+B+C=90º+C=180º; C=90º; 90º-A=30º, A=60º, B=30º;

4.- A+B+C=A+A-45º+A-30º=3A-75º=180º, A=85º, B=40º, C=55º
